# simform-task
This repository contains MEAN stack basic opertions

Get the Code

git clone https://github.com/mukeshannu94/simform-task.git

Start Front End
    1. cd user
    2. npm install
    3. ng serve --open

Start Back End
    1. cd api
    2. npm install
    3. npm start
